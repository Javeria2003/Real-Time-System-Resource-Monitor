# Real-Time-System-Resource-Monitor
Project Overview

The Real-Time System Resource Monitor is designed to continuously track and display system performance metrics with minimal overhead. It utilizes Windows performance counters and asynchronous updates to ensure smooth and accurate monitoring.

This project demonstrates practical knowledge of:

Object-Oriented Programming (OOP)

Multithreading & asynchronous programming

Performance monitoring APIs

Windows Forms / WPF development

Clean architecture principles
